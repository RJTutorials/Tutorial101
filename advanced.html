Advance Git Command

Command 1: git rebase

Explanation: Reapply commits on top of another base tip.

Example:

Rebase the current branch onto master:

$ git rebase master

Command 2: git cherry-pick

Explanation: Apply changes introduced by some existing commits.

Example:

Apply the changes introduced by the commit abc123:

$ git cherry-pick abc123

Command 3: git reflog

Explanation: Reference logs, or "reflogs", record when the tips of branches and other references were updated in the local repository.

Example:

Show the reflog for the HEAD reference:

$ git reflog

Command 4: git bisect

Explanation: Use binary search to find the commit that introduced a bug.

Example:

Start the bisecting process:

$ git bisect start

Command 5: git reset

Explanation: Reset current HEAD to the specified state.

Example:

Reset the current branch to the commit HEAD~2:

$ git reset HEAD~2

Command 6: git stash

Explanation: Stash the changes in a dirty working directory away.

Example:

Stash the changes:

$ git stash
